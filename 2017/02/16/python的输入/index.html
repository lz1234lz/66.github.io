使用optparse获取复杂命令行

一般情况下，我们使用optparse会使用到四个选项

  1. action
    action指明我们的指令的存储方式，一般有store sotre_true store_false三种情况

    1. sotre 就是将指令后方的参数值和指令关键字关联

    2. sotre_true 和store_false只与是否有这个指令相关，指令后方的值不影响指令的结果。

  2. dest
    dest是我们获取值的存储名，会根据action的设置获取到不同的值

  3. type
    type指定获取值的类型，有"string", "int", "long", "float", "complex", "choice"多种选项。

  4. help
    help就是我们指令的说明文档了
