python中的闭包

了解装饰器首先我们需要知道python的闭包

1、何为闭包

闭包就是能够读取其他函数内部变量的函数。例如在javascript中，只有函数内部的子函数才能读取局部变量，所以闭包可以理解成“定义在一个函数内部的函数“。在本质上，闭包是将函数内部和函数外部连接起来的桥梁。

装饰器

装饰器的作用

装饰器是程序开发中经常会用到的一个功能，一般会在这些情况下用到装饰器

  1. 引入日志
  2. 函数执行时间统计
  3. 执行函数前预备处理
  4. 执行函数后清理功能
  5. 权限校验等场景
  6. 缓存

总的来说呢，装饰器人如其名就是不影响函数的原方法，再给函数添加一个新的功能，不需要更改源代码。相当于给原来的函数做了一个装饰。

简单理解装饰器

@wapper相当于wapper(func)，这样一来， 即执行了simple_wapper函数执行之前或之后的功能，又执行了原来simple_wapper函数的内容，并将原simple_wapper函数返回值 返回

装饰器类别

无参装饰器

运行结果

say someting
run someting after say

包含return的装饰器

情况1，结果将会输出None
当包含return的情况下，会输出正常结果
do someting before

有参数的装饰器

固定参数装饰器

运行结果

无固定参数的装饰器

运行结果

k is k, v is 1
l = 1*1 + 2

运行结果
